# My movie recommender project by Kamsy Umeaku
This project suggests movies that are similar to a title entered by the user.  
It uses **content-based filtering** with **cosine similarity** on movie metadata (genres, cast, director, and keywords).  
Users can interact with the recommender through a clean web interface.

Core features:
- Enter a movie title to get similar recommendations  
- View poster images, ratings, and genres  
- Visualize movie trends by genre or year  
- Optional integration with OMDb or TMDb APIs  
- Fully deployable as a web app (Streamlit or Flask)
